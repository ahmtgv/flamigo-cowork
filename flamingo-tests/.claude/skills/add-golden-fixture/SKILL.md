# Добавление видео-фикстуры

1. Исходный `.mp4` положи в `tests/fixtures/source/`.
2. Добавь запись в массив `CLIPS` в `scripts/make-fixtures.sh`.
3. Зарегистрируй в `videoFixtures` в `tests/config/flamingo.ts` с ожидаемым
   **дискретным** состоянием.
4. Запусти `npm run fixtures`.
5. Проверь, что применился патч `C420mpeg2 → C420` — **без него Chrome падает**.
6. Держи 480p и ≤15 сек: Y4M сырой, ~1 ГБ на 10 сек 1080p.
7. Исходники версионируй через Git LFS, `.y4m` — в `.gitignore`.

## Политика
Каждый продовый инцидент с камерой в тот же день превращается в фикстуру.
Это то, что не даёт багу вернуться.
